# 🎹 Musitechne - Rhythm & Synth Game

Музыкальная браузерная игра. Играйте по готовым таймингам или создавайте уникальные треки!

## 👥 Команда разработчиков
* **Марк (TeamLead / Full Stack Developer):** Аудио-движок (Tone.js), математика Canvas, архитектура проекта.
* **Михаил (Core / Storage Developer):** Локальная БД (IndexedDB), управление пользовательскими данными `data/`, запись голоса `MediaRecorder API`.
* **Даниил (Frontend Developer):** UI/UX дизайн, HTML/CSS верстка, 2D-анимации.
* **Никита (QA / TechDocs):** Документирование, ручное тестирование, контроль качества деплоя.

* HTML5 / CSS3 / Vanilla JavaScript
* Звук: `Tone.js`
* Рендер (в разработке): `Canvas API`
* Хранение (в разработке): `IndexedDB` / `localStorage`

## 🕹 Как запустить?
Перейдите по ссылке в GitHub Pages и нажмите **"Включить звук"**, затем используйте клавиатуру для игры.
